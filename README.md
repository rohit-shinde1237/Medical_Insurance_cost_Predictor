# 🏥 Medical Insurance Cost Predictor

A Machine Learning project that predicts **medical insurance charges** based on an individual's demographic and lifestyle information. The application uses multiple regression models to estimate insurance costs, with **Random Forest Regression** selected as the final model due to its superior performance. The project is deployed using **Streamlit**, allowing users to receive instant predictions through a simple web interface.

---

## 📌 Project Overview

Medical insurance premiums are influenced by several factors, including age, BMI, smoking habits, number of children, gender, and region. This project analyzes these factors and builds predictive machine learning models capable of estimating insurance charges with high accuracy.

The project follows a complete machine learning pipeline:

* Data Collection
* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training
* Model Evaluation
* Model Deployment using Streamlit
